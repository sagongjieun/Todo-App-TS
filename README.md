# Todo-App-TS

## 프로젝트 소개
> React / TypeScript / vite / recoil 을 이용한 간단한 Todo App 입니다. ✅

## 프로젝트 목적
* TypeScript와 vite 환경에 익숙해지고자 프로젝트를 시작하게 되었습니다.
* 정말 간단한 앱이지만 recoil까지 적용해 selector의 사용법을 익혔습니다.
* Emotion에 props를 적용하는 법을 익혔습니다.
